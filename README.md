# 🌍 3D 互動地球儀應用程式

一個基於 Globe.gl 的互動式 3D 地球儀網頁應用，展示世界主要 100 個國家的地理資訊，並提供國歌播放功能。

![3D Globe Preview](https://img.shields.io/badge/Globe.gl-3D-blue) ![HTML5](https://img.shields.io/badge/HTML5-E34F26?logo=html5&logoColor=white) ![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?logo=javascript&logoColor=black)

## ✨ 主要功能

### 🗺️ 地理視覺化
- **國家邊界顯示**：清晰展示世界各國的地理邊界
- **首都標記**：紅色標記點標示 100 個主要國家的首都位置
- **3D 地球模型**：高品質的地球貼圖，包含地形凸起效果
- **大氣層效果**：逼真的大氣層視覺效果
- **夜空背景**：美麗的星空背景

### 🎯 互動功能
- **滑鼠拖曳旋轉**：自由旋轉地球查看不同角度
- **滾輪縮放**：使用滑鼠滾輪進行縮放
- **自動旋轉**：一鍵啟動地球自動旋轉動畫
- **點擊查看資訊**：點擊任何國家或首都查看詳細資訊
- **視角聚焦**：選擇國家後自動聚焦到該位置

### 🎵 多媒體功能
- **國歌播放**：點擊國家後可播放該國國歌（前 5 秒）
- **支援 16 個主要國家**的國歌資源

### 🌐 資訊顯示
- **繁體中文介面**：完整的繁體中文國家名稱
- **雙語標示**：同時顯示繁體中文和英文名稱
- **首都資訊**：顯示各國首都名稱

## 🚀 快速開始

### 線上演示
直接開啟 `index.html` 檔案即可使用，無需安裝任何套件！

1. 用瀏覽器開啟 `index.html`

## 🛠️ 技術架構

### 核心技術
- **HTML5**：網頁結構
- **CSS3**：樣式設計和動畫效果
- **JavaScript (ES6+)**：互動邏輯和資料處理

### 主要函式庫
- **[Globe.gl](https://github.com/vasturiano/globe.gl)** - 3D 地球視覺化引擎
- **[Three.js](https://threejs.org/)** - 3D 圖形渲染
- **Natural Earth Data** - GeoJSON 國家邊界資料

### 資料來源
- 國家邊界：[Natural Earth Vector](https://github.com/nvkelso/natural-earth-vector)
- 地球貼圖：Three-Globe 官方資源
- 國歌音檔：[Wikimedia Commons](https://commons.wikimedia.org/)

## 🔧 進階設定

### 離線使用
如需離線使用，可下載 JavaScript 函式庫到 `app/js/` 目錄：

1. 下載 Three.js：`three.min.js`
2. 下載 Globe.gl：`globe.gl.min.js`

然後修改 `index.html` 中的腳本引用：
```html
<script src="app/js/three.min.js"></script>
<script src="app/js/globe.gl.min.js"></script>
```

### 自訂初始視角
修改第 1170 行的初始視角設定：
```javascript
world.pointOfView({ lat: 25.033, lng: 121.5654, altitude: 2.5 }, 1000);
```

### 新增更多國家國歌
在 `anthemUrls` 物件中新增國家代碼和音檔 URL：
```javascript
const anthemUrls = {
  'TW': 'https://...',
  // 新增更多國家...
};
```
